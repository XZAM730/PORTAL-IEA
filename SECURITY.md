# Security Policy

## Supported Versions

Sistem Portal IEA berikut yang masih didukung pembaruan keamanannya:

| Version | Supported          |
| ------- | ------------------ |
| 4.0.x   | :white_check_mark: |
| 3.0.x   | :x:                |
| < 3.0   | :x:                |

## Reporting a Vulnerability

Jika Anda menemukan celah keamanan (bug/exploit) pada sistem portal ini, mohon **JANGAN** membukanya di Issue Tracker publik.

Silakan laporkan langsung kepada System Architect:

* **Email:** [Email-Kamu-Di-Sini]
* **Subject:** `[SECURITY ALERT] IEA Portal Vulnerability`

Kami akan merespons dalam waktu 2x24 jam. Terima kasih telah menjaga keamanan data IEA.
